# MODULY

Hackathon build of a browser-based desktop simulator.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Login

Default accounts are created by the simulator filesystem.

- `root` password: `admin`
- `user` password: `1234`
- `guest` password: `guest`

## Notes

The app keeps the simulator features: window manager, dock, settings, file manager, terminal, browser, app store, music, photos, mail, messages, users, storage reset, and wallpaper/accent controls. This copy uses `src/assets/images/images.jpeg` as the default wallpaper.

The project remains private for hackathon use. License metadata is retained in the source tree.
